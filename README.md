## hwat, hwy & hwere

base tooling & scripts to build & run the shelly repo focusing on a reproducible linux installation.
this will clear & overwrite config files for rioterm, nvim & nushell

inside repo run:

`nu ./init.nu`

----

# Pre-req (wip)
- git
- cargo (incl. rustup & rustc)
- nu
- node
- rioterm (optional)
